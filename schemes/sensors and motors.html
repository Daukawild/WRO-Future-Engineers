Lesson 3.1: Introduction to LEGO Mindstorms EV3 Sensors
Class Discussion / New Material

Topic #3:  Robot Sensors
Learning Goal #3: Student will be able to use sensors appropriately on a robot.


Item #1: Robot Sensors
Sensors are used to measure a physical quantity, and then convert it into a usable signal that can be used by your robot.  When designing your robot it is important for you to choose the correct sensor/s so that your robot will be able to function properly in its operating environment.  For example if you need your robot to sense an obstacle such as a wall you would want to use a Touch Sensor.  By adding the proper sensor/s, your robot will be aware of its environment to the degree necessary to perform the assigned tasks.  Consequently, during the design process you must consider the environment in which your robot will operate along with its intended function and the required level of precision and accuracy, and only then select the required sensors.


Item #2: LEGO Mindstorms EV3 Sensors
View and learn what each of the EV3 Sensors looks like and what its function is.



Sound Sensor:
The Sound Sensor allows your LEGO Robot to "hear".  The Sound Sensor is able to measure noise levels in decibels (dB) and frequencies around 3 - 6 kHz (dBA) which is where the human ear is most sensitive.
Color Sensor:
The Color Sensor can detect the color of a surface (Color Mode), the amount of light reflected by a surface (Reflected Intensity Mode), or the brightness of ambient light (Ambient Light Intensity Mode).  You can use the Color Sensor to detect the color of a surface, follow lines or respond to light signals.
Ultrasonic Sensor:
The Ultrasonic Sensor generates sound waves and reads their echo returns to detect and measure the distance to objects.  The Ultrasonic Sensor can function as a sonar by sending out single sound waves. It can also listen for a sound wave which can be used to trigger programed event/s.
Touch Sensor:
The Touch Sensor allows you to "feel" by detecting whether the red button on the sensor is currently pressed or released.  The EV3 retrieves this information and can use it in programs.  For example, you could program your robot to activate claws whenever the Touch Sensor is pressed.
Gyro Sensor:
The Gyro Sensor measures the robot's rotational motion and changes in its orientation.  You can use the Gyro Sensor to measure angles, create balancing robots and explore technology associated with navigation systems, game controllers and gyroscopic stabilized platforms.
Infrared Sensor (Home Version):
The Infrared Sensor allows you to "see" its surroundings by measuring the approximate distance to an object using infrared light.  The Infrared Sensor gathers information using the infrared remote (beacon).  It can: 1) detect which buttons on the remote you press; 2) how far away the remote is; and the relative direction from the robot to the remote.
Rotation Sensor:
The Rotation Sensor determines how many robot wheel rotations have been made and can be programmed to stop the robot wheels after a specified number of rotations.

Item #3: LEGO Mindstorms EV3 Ports
All of these sensors, with the exception of the Rotation Sensor, are inputs to the LEGO Mindstorms EV3 and therefore should be connected to ports 1, 2, 3 or 4.  The Rotation Sensor is associated with the motors and is therefore an output of the LEGO Mindstorms EV3 and should be connected to ports A, B, C or D.
